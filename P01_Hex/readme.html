Implementations =================== HexBoard is a well known two player board game. The goal of this game to make a connection between the players two sides. That is what the implementation of `HexBoard.java` is trying to model. `HexBoard.java` is an object class for the game. There are many different functions in this class that allows the players to receive information about their game progress. These include isSet(), getPlayer(), setTile(), numberOfUnsetTiles(), hasPlayer1Won(), hasPlayer2Won(), and isOnWinningPath(). In my implementation I had to us an object called union-find. This allows the user an easy way to check if a player has a connected path to the player's two virtual nodes.
